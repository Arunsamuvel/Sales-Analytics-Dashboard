# Sales Analytics Dashboard
📊 Project Overview
The Sales Analytics Dashboard is an interactive Excel dashboard designed to provide a comprehensive view of sales performance, customer behavior, profitability, and service effectiveness. The dashboard enables business stakeholders to monitor key performance indicators (KPIs), identify trends, analyze customer acquisition channels, and make data-driven decisions.
This project demonstrates advanced Excel dashboarding techniques using Pivot Tables, Pivot Charts, Slicers, KPI Cards, and Data Visualization.

🎯 Business Problem
Organizations often struggle to consolidate sales data from multiple sources and transform it into actionable insights.
This dashboard helps answer critical business questions such as:


How much revenue has been generated?


What is the overall profit margin?


How many new and repeat customers were acquired?


Which customer acquisition channel performs best?


Which city contributes the highest sales?


Which service generates the most revenue?


Who are the top customers?


How does sales performance change over time?
